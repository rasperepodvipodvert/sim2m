# Sim2M Parser

Парсер данных о трафике с сайта sim2m.ru через API.

## Установка

```bash
python3 -m venv venv
source venv/bin/activate
pip install requests
```

## Быстрый старт

### 1. Получите токен из браузера

1. Откройте https://pay.sim2m.ru и войдите
2. F12 → Application → Cookies → `clientAppRefreshToken`
3. Скопируйте значение

### 2. Создайте файл `.env`

```bash
REFRESH_TOKEN=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...
ACCOUNT_ID=95143
```

### 3. Запустите парсер

```bash
source venv/bin/activate
python parser.py
```

## Использование

### С файлом .env (рекомендуется)

```bash
python parser.py
```

### С параметрами

```bash
python parser.py 'ваш_токен' 95143
```
